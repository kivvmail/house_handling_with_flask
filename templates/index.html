Назначение:

Программа предназначена для парсинга сведений о квартирах в доме по заданному адресу и обработки данных для создания плана размещения квартир в доме по подъездам и этажам.

Инструкции:

  • Перейдите в раздел "Новый дом".
  • Последовательно выберите выберите макрорегион, регион и населенный пункт из списка.
  • Введите в соответствующие поля название улицы, номер дома, номер корпуса(если есть).
  • Нажмите кнопку "Запустить парсинг", повторите через некоторое время при сообщении о неудаче.
  • При успешном завершении парсинга нажмите кнопку "Заупстить обработку данных".
  • После вывода на экран плана дома по кнопке "Редактировать" доступно назначение для заданной квартиры произвольных значений номера этажа и номера позиции на этаже.
  • Список ранее обработанных домов находится в разделе "Обработанные дома".
